# Setup

```bash
npm install
cd workspaces/puth && bun run dev:browser:install && cd -
```

# Development

```bash
bun dev
```

## Puth Dev Server

```bash
bun run dev:server
```

## Puth Dev GUI

```bash
bun run dev:client
```
